Embedded systems:

Raspberry pi pico:
a micro-controller board used in a widerange of applications and contains GPIO pins which can be used to control and recieve inputs from electronic devices. The Raspberry Pi Pico's MCU processes incoming data and commands, and then uses its GPIO pins to control external devices and sensors.
